// DIAGNOSTIC PROTOCOL

Four Steps to Clean PRs

01

Scan

Run pr-surgeon scan. Reads your git diff automatically. Detects all changed files, content, and import chains.

02

Diagnose

5-dimensional analysis: cognitive complexity, domain classification, coupling graph, risk assessment, and size metrics.

03

Split

Smart algorithm groups by domain, respects import coupling, assigns merge order. Each suggested PR is self-contained.

04

Ship

Follow the plan. Ship focused PRs. Get faster reviews, fewer bugs, happier teammates. See before/after proof.

// GET STARTED

Running in 30 Seconds

No config. No API keys. No setup wizards. Just scan and go.

⚡

Install

# Clone & install
$ git clone https://github.com/hamza0281/pr-surgeon.git
$ cd pr-surgeon
$ npm install

# Run demo (no git repo needed)
$ npm run demo
🔍

Scan

# Scan current changes
$ node src/cli.js scan

# Against a specific branch
$ node src/cli.js scan -b develop

# Generate HTML X-Ray report
$ node src/cli.js scan --report
🛠

Automate

# Install pre-push hook
$ node src/cli.js hook install

# Analyze team history
$ node src/cli.js history

# Full HTML report
$ node src/cli.js report
