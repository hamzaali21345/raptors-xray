GET STARTED

Running in 30 Seconds

No config files. No API keys. No setup wizards. Clone, install, scan.

01 Install

# Clone & install $ git clone https://github.com/hamza0281/pr-surgeon.git $ cd pr-surgeon $ npm install

02 Scan

# Scan current changes $ node src/cli.js scan # With HTML report $ node src/cli.js scan --report

03 Automate

# Auto-scan before every push $ node src/cli.js hook install # Run demo (no repo needed) $ npm run demo
